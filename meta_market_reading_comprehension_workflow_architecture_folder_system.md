# Meta-Market Reading Comprehension Skills Thinking Workflow Architecture

## Purpose
Create a reusable folder architecture and cognitive workflow for reading, decoding, interpreting, and synthesizing market information across financial, economic, geopolitical, technological, ESG, and behavioral domains.

The architecture is designed to improve:
- Signal detection
- Context interpretation
- Cross-market pattern recognition
- Narrative decomposition
- Decision-quality thinking
- Strategic reading capability under uncertainty

## Five-Layer Reading Model

### Layer 1: Literal Reading
Question:
- What does the text explicitly say?
- What are the reported numbers, events, dates, and actors?

Output:
- Raw facts
- Key metrics
- Direct statements

### Layer 2: Analytical Reading
Question:
- Why did this happen?
- What assumptions are embedded?
- Which variables changed?

Output:
- Drivers
- Mechanisms
- Initial interpretation

### Layer 3: Systemic Reading
Question:
- How does this connect with other systems?
- What are the cross-sector or macro implications?

Output:
- Network effects
- Cross-market transmission
- System map

### Layer 4: Strategic Reading
Question:
- What matters most?
- What future scenarios become more probable?

Output:
- Scenario ranking
- Opportunity-risk map
- Strategic implications

### Layer 5: Meta-Reading
Question:
- What may be missing?
- Which biases may distort interpretation?
- What would falsify the current view?

Output:
- Contradictory evidence
- Blind spots
- Revised confidence level

### Core System Logic

```text
Input Variable
→ Market Mechanism
→ Intermediate Effects
→ Feedback Loop
→ Emergent Outcome
```

### Example Feedback Loops

#### Reinforcing Loop

```text
Rising Asset Prices
→ Increased Investor Confidence
→ More Buying
→ Further Price Increase
```

#### Balancing Loop

```text
High Inflation
→ Higher Interest Rates
→ Lower Demand
→ Reduced Inflation
```

### Nonlinear Dynamics
Market systems rarely change linearly. Small changes can remain invisible until a threshold is crossed.

Example:

```text
Debt Increase
→ Stable System
→ Threshold Reached
→ Sudden Credit Stress
→ Rapid Market Repricing
```
