# Workflow Architecture: Meta-Capability – Understand System Thinking

## Purpose
Create a reusable folder architecture for developing the capability to understand systems thinking across individual, organizational, technological, and civilizational scales.

The architecture is organized from foundational concepts toward advanced system dynamics, cross-scale integration, and practical application.

# Meta-Capability Output

The completed architecture should enable four higher-order capabilities:

1. Understand why systems behave as they do
2. Detect hidden causes and delayed consequences
3. Anticipate emergent and nonlinear outcomes
4. Design more effective interventions across scales
